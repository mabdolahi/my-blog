# my-blog
Simple blog site with class comment

To start project:


WINDOWS:
```bash
pip install virtualenv
```
```bash
python -m venv myenv
```
cd myenv/scripts

type 'activate'

cd my_site

python manage.py runserver



MAC:

$ pip3 install virtualenv

$ virtualenv venv

$ source venv/bin/activate

cd my_site

python manage.py runserver
